About the Crime Map +

This map is the new and improved version of the original which is located at the Police.co.uk website. The Crime Map+ is populated with data obtained from the crimes API and submitted by local forces.

What's New?

Previously, users were unable to visualise the severity of each crime in the original map. But with the Crime Map + users are able to see a variety of different crime severities. The crime severity key can be seen below or by viewing the navigation bar.
